* add two button:

_Find in Current Tab_ button: 
It finds keywords in the active document.
It ignores `files`, `skip`, `from`, `depth` input field.

_Find in All Tabs_ button: 
It finds keywords in all opened document.
It ignores `files`, `skip`, `from`, `depth` input field.
